Modern Witches Confluence Rebrand

Fall 2019

The Modern Witches Confluence takes place annually in San Francisco, California. This project is a rebrand of the confluence's print materials: brochure/poster, billboard, banner, book, and logo. Witchcraft can have archaic stereotypes tied to them, which does not represent every witch's practices. The rebrand needed to embody all types of witches. The calm clean color palette gives the conference an updated look, appealing to every modern witch.
